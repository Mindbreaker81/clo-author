# Reporting Guidelines Reference

Quick reference for the writer, writer-critic, domain-referee, methods-referee, and `/review --reporting` flows.

## How to Use This File

1. Identify the primary study design.
2. Apply the matching guideline first.
3. If the paper mixes designs, use the stricter combination (for example STROBE + STARD for an observational diagnostic study).
4. Missing ethics, registration, outcome definitions, harms, or flow/accounting of participants should be treated as major problems.

---

## CONSORT 2010 — Randomized Controlled Trials

| Item(s) | Section | What to Confirm |
|---------|---------|-----------------|
| 1a-1b | Title / Abstract | Identify the study as randomized; provide a structured summary |
| 2a-2b | Introduction | Scientific background and specific objectives / hypotheses |
| 3a-3b | Methods: trial design | Trial design, allocation ratio, and important post-start changes |
| 4a-4b | Methods: participants | Eligibility criteria and study setting |
| 5 | Methods: interventions | Interventions detailed enough to replicate |
| 6a-6b | Methods: outcomes | Pre-specified primary/secondary outcomes and assessment timing |
| 7a-7b | Methods: sample size | Sample-size logic and any interim stopping guidance |
| 8a-10 | Methods: randomization | Sequence generation, concealment, implementation |
| 11a-11b | Methods: blinding | Who was blinded and how |
| 12a-12b | Methods: statistics | Primary/secondary analyses and additional analyses |
| 13a-13b | Results: participant flow | Numbers randomized, treated, analyzed, and losses/exclusions |
| 14-16 | Results: recruitment / baseline / numbers analyzed | Recruitment dates, baseline table, denominators for analysis |
| 17a-19 | Results: outcomes / ancillary / harms | Effect estimates with precision, subgroup caution, adverse events |
| 20-22 | Discussion | Limitations, generalizability, interpretation consistent with harms/benefits |
| 23-25 | Other information | Registration, protocol access, funding |

## STROBE — Observational Studies

| Item(s) | Section | What to Confirm |
|---------|---------|-----------------|
| 1a-1b | Title / Abstract | Study design named explicitly; balanced summary provided |
| 2-3 | Introduction | Background, rationale, and objectives |
| 4-5 | Methods: design / setting | Key design elements and relevant dates/settings |
| 6 | Methods: participants | Eligibility, selection, and follow-up / matching details |
| 7-8 | Methods: variables / data sources | Outcomes, exposures, confounders, effect modifiers, measurement methods |
| 9 | Methods: bias | Main sources of bias and mitigation strategy |
| 10 | Methods: study size | Rationale for sample size or available population |
| 11-12 | Methods: quantitative variables / statistics | Variable handling, confounding adjustment, subgroup and sensitivity plans |
| 13 | Results: participants | Flow of participants and reasons for exclusions |
| 14-15 | Results: descriptive / outcome data | Descriptive data, missingness, outcome counts or summaries |
| 16-17 | Results: main / other analyses | Effect estimates with precision, adjusted vs unadjusted, subgroup or sensitivity work |
| 18-21 | Discussion | Key results, limitations, bias direction/magnitude, interpretation, generalizability |
| 22 | Other information | Funding and role of funders |

## PRISMA 2020 — Systematic Reviews and Meta-analyses

| Item(s) | Section | What to Confirm |
|---------|---------|-----------------|
| 1-2 | Title / Abstract | Review identified as systematic review; structured abstract present |
| 3-4 | Introduction | Rationale and explicit objectives / questions |
| 5-8 | Methods: eligibility / sources / search / selection | Inclusion criteria, sources searched, full strategy, screening process |
| 9-11 | Methods: data collection / items / risk of bias | Extraction process, collected variables, bias assessment tools |
| 12-15 | Methods: effect measures / synthesis | Effect measures, synthesis methods, heterogeneity handling, sensitivity plans |
| 16 | Results: study selection | PRISMA flow with counts and exclusions |
| 17-19 | Results: characteristics / bias / individual studies | Included study characteristics, bias judgments, study-level findings |
| 20-22 | Results: syntheses / reporting bias / certainty | Pooled results, heterogeneity, publication bias, certainty assessment |
| 23 | Discussion | Interpretation in light of evidence, limits, implications |
| 24-27 | Other information | Registration/protocol, support, competing interests, data/material availability |

## STARD 2015 — Diagnostic Accuracy Studies

| Item(s) | Section | What to Confirm |
|---------|---------|-----------------|
| 1-2 | Title / Abstract | Diagnostic accuracy study identified; structured summary present |
| 3-4 | Introduction | Clinical role of the index test and study objectives |
| 5-9 | Methods: design / participants | Prospective vs retrospective design, eligibility, recruitment basis, setting, dates |
| 10a-13b | Methods: test methods | Index test, reference standard, rationale, cutoffs, blinding |
| 14-18 | Methods: analysis | Accuracy measures, indeterminate/missing handling, variability analyses, sample size |
| 19 | Results: participant flow | Flow diagram and participant accounting |
| 20-23 | Results: baseline / disease severity / test distribution | Clinical characteristics, alternative diagnoses, timing, cross-tabulation |
| 24-25 | Results: accuracy estimates / adverse events | Sensitivity, specificity, precision, complications from testing |
| 26-27 | Discussion | Limitations, intended use, implications for practice |
| 28-30 | Other information | Registration, protocol access, funding |

## Notes for Reviewers

- Use CONSORT for randomized therapeutic or procedural trials.
- Use STROBE for cohort, case-control, and cross-sectional observational work.
- Use PRISMA for evidence synthesis and meta-analysis.
- Use STARD for diagnostic accuracy studies.
- Consider TRIPOD expectations for prediction models even though the full checklist is not reproduced here.
